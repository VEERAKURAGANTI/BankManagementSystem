# 🏦 Bank Management System (Java Console Application)

This is a simple **Bank Management System** implemented as a Java console application.  
It allows managing customers and their multiple bank accounts with functionalities like creating accounts, depositing, withdrawing money, and viewing balances.

---

## ✨ Features

- Add and remove customers
- Create multiple accounts per customer
- Deposit money into accounts
- Withdraw money from accounts
- Show account balances per customer
- Display customer details with account info
- Menu-driven console interface

---

## 🛠️ Technologies Used

- Core Java (OOP Concepts)
- Collections Framework (HashMap)
- Scanner for console input
